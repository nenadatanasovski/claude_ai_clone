2. `showOnboardingTour` checks localStorage for 'onboardingCompleted'

4. User can navigate through 6 steps with Next/Back
5. User can jump to specific step by clicking progress dots
6. User can skip tour at any time
7. Completing or skipping sets localStorage flag
8. Tour never shows again for that user
9. Subsequent visits skip tour, show main app
