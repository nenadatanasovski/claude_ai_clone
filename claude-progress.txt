**Feature #11: Auto-resize Textarea as Content Grows** ✅ COMPLETED

Implementation Details:

**Frontend Changes (src/App.jsx):**

1. Added useEffect hook for auto-resize (lines 86-96):
   ```javascript
   useEffect(() => {
     const textarea = textareaRef.current
     if (textarea) {
       // Reset height to minHeight to get the correct scrollHeight
       textarea.style.height = '52px'
       // Calculate new height based on scrollHeight
       const newHeight = Math.max(52, Math.min(textarea.scrollHeight, 300))
       textarea.style.height = `${newHeight}px`
     }
   }, [inputValue])
   ```

2. Updated textarea element (lines 531-543):
   - Removed fixed `rows="2"` attribute
   - Added `overflow-hidden` class to prevent scrollbar during expansion
   - Added inline `style={{ minHeight: '52px' }}` for base height
   - Kept `resize-none` class to prevent manual resizing

**How It Works:**
- Effect triggers whenever inputValue changes
- First resets height to 52px to get accurate scrollHeight measurement
- Calculates new height: min 52px, max 300px, based on content
- Textarea grows smoothly as user types
- Shrinks back when content is deleted
- Becomes scrollable if content exceeds 300px
