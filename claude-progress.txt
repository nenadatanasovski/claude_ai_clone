**Animation Details:**

The `animate-slideIn` class uses the existing CSS animation defined in index.html (lines 55-64):

```css
@keyframes slideIn {
  from {
    opacity: 0;
    transform: translateY(-20px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}

.animate-slideIn {
  animation: slideIn 0.3s ease-out;
}
```

**Animation Properties:**
- Duration: 300ms (0.3s)
- Easing: ease-out (smooth deceleration)
- Effects: Fade in (opacity 0 → 1) + Slide down (translateY -20px → 0)
- Visual result: Dropdown gracefully fades in while sliding down from above
